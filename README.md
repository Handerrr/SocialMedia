# SocialMedia API 

Uma rede social completa desenvolvida com Django REST Framework no backend e React + TypeScript no frontend.

O projeto possui sistema de autenticação JWT, feed de posts, comentários, likes, seguidores e upload de imagens utilizando Cloudinary.

---

- Login com JWT Authentication
- Logout
- Perfil do usuário
- Editar perfil
- Upload de foto de perfil
- Biografia personalizada
